# Skywalk Learning Game - React Version

Game học tập về Chủ nghĩa xã hội khoa học được xây dựng bằng React.js

## Cài đặt

```bash
npm install
```

## Chạy ứng dụng

```bash
npm start
```

Ứng dụng sẽ chạy trên [http://localhost:3000](http://localhost:3000)

## Build cho production

```bash
npm run build
```

## Tính năng

- ✅ Hệ thống điểm số và thống kê
- ✅ Hiệu ứng animation đẹp mắt
- ✅ Thanh tiến trình
- ✅ Phản hồi ngay lập tức
- ✅ Màn hình hoàn thành với tổng kết

- React 18
- React Hooks (useState, useEffect)
